Welcome to the AiiDAlab deployment for PSI tutorials!

This AiiDAlab server is deployed and maintained by the Materials Software and Data Group (MSD) and its aim is to serve users during the course Electronic-structure simulations for user communities at large-scale facilities.

The virtual machines used for this course are provided by the Azure cloud service.

Login to AiiDAlab

The service is accessible with GitHub credentials:

  • Click on the "Sign in with GitHub" button.
  • Authorize the application to access your GitHub account (needed for authentication).
  • Once you are logged in, you will be redirected to the AiiDAlab home page.

Getting started with AiiDAlab

AiiDAlab is a web environment to run materials simulation online. It is based on Jupyter as the web backend, and AiiDA as the workflow engine.

Some resources to learn AiiDAlab:

  • Our official documentation, for a more detailed introduction and hands on guide.
  • An AiiDAlab introduction video for an overview.

Any questions?

Please checkout our FAQ page. If you cannot find your question, feel free to send an email to the developers. For technical issues, open an issue on GitLab and we will try to resolve it.
